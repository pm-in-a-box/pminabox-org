PM in a Box marketplace is the operator-facing catalog for community and vendor-contributed agents, skills, tools, and connectors. Browse what is available, see what each listing depends on, and add the right capabilities to your box.

How the catalog works

Four building blocks, one install flow.

Agents

Named teammates operators can @mention in chat, like Lisa the leasing agent. Agents usually bundle multiple skills and declare every tool or connector they need.

View agents →
Skills

Markdown-based capabilities that describe workflows, knowledge, and instructions. Skills can require tools and connectors, but they are not the main @mention surface for most operators.

View skills →
Tools

General-purpose actions that agents and skills can use: PDF extraction, spreadsheet manipulation, browser automation, document redaction, and approval workflows.

View tools →
Connectors

External data and system access, usually MCPs: PMS platforms, email, calendars, accounting, phones, maintenance systems, leasing tools, and more.

View connectors →
